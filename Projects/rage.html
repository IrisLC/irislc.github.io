Rage Game, Spring 2024

Unreal Engine

The final project for a Game Development class, tasked with creating a 3d platformer. Set out to make a rage game that achieves this effect via traps that the player cannot see coming (in the vein of 2d games like "I want to be the guy" and "Cat Mario") and using player's expectations against them. First experience with using Unreal Engine 5 Blueprints.
